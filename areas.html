SisControlA: Sistema de controle de acesso de baixo custo

Projeto para apresentar o sistema de controle de acesso de baixo custo (SisControlA), um sistema alternativo, microcontrolado por uma placa Arduino, que se baseia na identificação de cartões ou tags Radio-frequency identification para acionar um fecho magnético, liberando a entrada do usuário ao ambiente.
